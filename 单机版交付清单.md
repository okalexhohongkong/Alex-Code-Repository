# 黑卫士 AI 数字档案管理系统单机版交付清单

## 当前版本定位

当前版本是“可安装/可演示 + 单机索引闭环”的第一版。它已经可以启动页面、扫描样本目录、生成本地索引、读取真实索引、做基础检索、生成验收报告和备份快照，并且已纳入多源存储接入字段和同步方案。

多源归集的目标已经明确：手机、iPad、旧电脑、邮箱、云端、NAS、U盘、SD卡、相机、摄像机都要能纳入统一管理，最终以 NAS 作为归总硬件。当前单机版先做只读索引和演示闭环，不擅自移动、删除、改名真实文件；授权后才进入完整采集到 NAS 暂存区，再分类、去重、索引和人工复核。

## 启动命令

| 用途 | 命令 | 说明 |
|---|---|---|
| 桌面打开主系统 | 双击桌面 `黑卫士AI数字档案管理系统.command` | 自动启动或复用本地服务，并打开首页 |
| 桌面打开进度看板 | 双击桌面 `黑卫士AI项目进度看板.command` | 自动启动或复用本地服务，并打开甘特图看板 |
| 桌面只读扫描文件夹 | 双击桌面 `黑卫士AI只读扫描文件夹.command` | 选择一个文件夹，只读扫描并更新索引、验收报告和备份 |
| 桌面打开项目文件 | 双击桌面 `黑卫士AI数字档案管理系统-项目文件` | 进入项目文件夹，不搬动项目主体 |
| 一键演示 | `npm run demo` | 自动扫描、验收、自检、备份并启动服务 |
| 只启动页面 | `npm run dev` | 启动本地演示服务 |
| 命令行打开主系统 | `npm run open` | 自动检查服务并打开首页 |
| 命令行打开进度看板 | `npm run board` | 自动检查服务并打开进度看板 |
| 命令行选择扫描目录 | `npm run scan:choose` | 弹出文件夹选择器后只读扫描 |
| 扫描当前目录 | `npm run scan` | 只读扫描当前项目目录 |
| 扫描指定目录 | `npm run scan -- /文件夹路径` | 只读扫描指定样本档案目录 |
| 合并多个索引 | `npm run merge -- 索引A.json 索引B.json` | 多目录扫描后合并去重，生成脱敏合并索引 |
| 单机验收 | `npm run verify` | 生成验收报告 |
| 备份快照 | `npm run backup` | 备份索引、看板和验收报告 |
| 服务健康检查 | `npm run health` | 检查首页、看板、健康接口和浏览器脱敏索引 |
| 演示自检 | `npm test` | 检查关键文件和核心入口 |

## 已具备能力

- 大屏首页和一级菜单。
- 项目、公司、部门、员工、人事职能检索。
- 公司类型、公司名称、部门建制、项目类型、负责人等级检索。
- 模糊时间段的周期搜索和按周期阶段模糊搜索。
- L0 外部流通、L1 普通、L2 内部、L3 敏感、L4 机密、L5 最高授权、L6 绝密。
- 完整状态：残缺待补充、残缺已补充、半成品待收尾、完整已归档、待人工确认。
- 作品等级：常规作品、优质作品、优秀案例、经典案例、示范文档、惯例制度、经典制度。
- 图片、视频、录音、邮件、合同、PPT、Word、Excel、扫描件、标识文件、方案、标书等作品类型入口。
- 本机索引快搜。
- 真实本机索引读取。
- 文件相对路径、大小、修改时间展示。
- 多源存储检索：手机、iPad、旧电脑、邮箱、本机硬盘、在线硬盘、外接硬盘、U盘、SD卡、相机、摄像机、云端云盘、公有云、私有云、局域网网盘、NAS、不同主机硬盘、同主机多接口硬盘。
- 同步策略展示：只读索引、NAS 暂存区、云本同步、NAS 主库、私有云同步、公有云冷备、三方交叉校验。
- 存储安全字段展示：接入方式、同步状态、交叉校验、存储风险、成本等级。
- 复制相对路径和按档案编号打开所在位置申请入口。
- 颜色、字体、字号自定义。
- 自检、验收、备份、进度看板。
- 桌面主系统入口、桌面进度看板入口、桌面只读扫描文件夹入口、桌面项目文件入口。
- 扫描批次报告：每次扫描生成批次编号、分布统计、安全边界和脱敏浏览器摘要。
- 批次历史看板：按批次展示来源、数量、体积、默认密级、主要类型、风险和合并索引摘要。
- 多目录合并索引：支持多个扫描索引合并去重，生成浏览器脱敏合并索引。
- 服务健康检查：确认首页、看板、健康接口和浏览器脱敏索引正常。

## 下一版建议

- 增加真实样本硬盘目录选择。
- 增加 SQLite 本地数据库，支撑几十万到百万级文件索引。
- 增加多目录合并索引，支持本机硬盘、NAS、云盘和跨主机索引统一汇总。
- 增加 NAS 暂存区流程，支持授权后把手机、iPad、旧电脑、邮箱、U盘、SD卡、相机、摄像机资料分批采集到 NAS。
- 增加云端清单/NAS 清单对账，自动发现离线、重复、缺失和版本变化。
- 增加权限矩阵：查看、复制、下载、修改、删除、导出。
- 增加 OCR、录音转写、视频抽帧、图片人脸识别候选和 AI 喂料清单。
- 增加未完成作品的 AI 收尾流程和评分标准。

安全边界：L4、L5、L6 不进公有云，不进入 AI 训练；图片人脸识别候选必须受权限控制。
